# Contributor Covenant 行為準則

## 我們的承諾

為了營造開放友善的環境，我們以參與者與維護者的身分承諾：讓每個人參與本專案與社群的經驗免於騷擾，無論年齡、體型、身心障礙、族裔、性別認同與表達、經驗程度、國籍、外貌、種族、宗教或性傾向。

## 我們的準則

有助於正向環境的行為包括：展現同理與善意、尊重不同意見與經驗、給予並優雅接受建設性回饋、為錯誤負責並道歉、以社群整體利益為重。

不可接受的行為包括：性化的言語或意象與任何形式的性騷擾、辱罵或貶損性留言與人身或政治攻擊、公開或私下的騷擾、未經明確許可公開他人私人資訊。

## 執行

可接受與不可接受行為的標準由維護者負責澄清與執行，維護者有權移除、編輯或拒絕不符本準則的留言、commit、程式碼、issue 與其他貢獻。

## 檢舉

濫用、騷擾或其他不可接受的行為可向專案維護者回報：**<your-email@example.com>**。所有申訴都會被審視與調查，並做出適當且必要的回應。
